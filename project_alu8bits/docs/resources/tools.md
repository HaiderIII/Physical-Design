# Tools Used in This Project

## Primary Tools

### OpenROAD
- **Purpose**: Complete RTL-to-GDSII flow
- **Version**: v2.0
- **Website**: https://theopenroadproject.org/
- **Documentation**: https://openroad.readthedocs.io/
- **Description**: Open-source physical design tool for digital circuits

### Yosys
- **Purpose**: RTL synthesis
- **Version**: 0.60
- **Website**: https://yosyshq.net/yosys/
- **Documentation**: https://yosyshq.readthedocs.io/
- **Description**: Framework for RTL synthesis and formal verification

### Magic
- **Purpose**: Layout viewer, DRC, LVS
- **Version**: 8.2.157
- **Website**: http://opencircuitdesign.com/magic/
- **Documentation**: http://opencircuitdesign.com/magic/userguide.html
- **Description**: VLSI layout tool for viewing and editing layouts

### KLayout
- **Purpose**: GDSII/OASIS viewer and editor
- **Version**: 0.26.2
- **Website**: https://www.klayout.de/
- **Documentation**: https://www.klayout.de/doc/index.html
- **Description**: High-performance layout viewer and editor

## Process Design Kit (PDK)

### SkyWater Sky130
- **Technology**: 130nm
- **Type**: Open-source PDK
- **Version**: 2024.08.17
- **Website**: https://skywater-pdk.readthedocs.io/
- **Description**: First open-source foundry PDK
- **Standard Cells**: sky130_fd_sc_hd (High Density)

## Utility Tools

### Volare
- **Purpose**: PDK version manager
- **Description**: Tool to download and manage different PDK versions

### Git
- **Purpose**: Version control
- **Description**: Track changes and collaborate on design files
