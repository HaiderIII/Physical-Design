# Recommended Books for Physical Design

## Essential Reading

### 1. CMOS VLSI Design: A Circuits and Systems Perspective
- **Authors**: Neil Weste, David Harris
- **Description**: Comprehensive introduction to CMOS design, covers physical design fundamentals
- **Topics**: Logic design, circuit design, physical design, timing analysis

### 2. Physical Design of CMOS Integrated Circuits Using Cadence Tools
- **Authors**: Jitendra Kumar Mohan, Pradeep Kumar Sharma
- **Description**: Practical guide to physical design flow
- **Topics**: Floorplanning, placement, routing, timing closure

### 3. Electronic Design Automation for IC Implementation, Circuit Design, and Process Technology
- **Editor**: Luciano Lavagno, Igor Markov, Grant Martin, Louis Scheffer
- **Description**: Comprehensive EDA reference
- **Topics**: Synthesis, placement, routing, verification

### 4. VLSI Physical Design: From Graph Partitioning to Timing Closure
- **Authors**: Andrew B. Kahng, Jens Lienig, Igor L. Markov, Jin Hu
- **Description**: Academic perspective on physical design algorithms
- **Topics**: Algorithms for partitioning, floorplanning, placement, routing

## Additional Resources

### 5. Digital Integrated Circuit Design Using Verilog and Systemverilog
- **Authors**: Ronald W. Mehler
- **Description**: RTL design and synthesis
- **Topics**: Verilog, synthesis, timing analysis

### 6. Static Timing Analysis for Nanometer Designs: A Practical Approach
- **Authors**: J. Bhasker, Rakesh Chadha
- **Description**: Deep dive into timing analysis
- **Topics**: STA fundamentals, clock domain crossing, timing constraints
